# Todo List App

- A simple todo list application that allows users to add, complete, and delete tasks.

### Features

- Add new tasks to the list.
- Mark tasks as complete.
- Delete tasks from the list.
- Responsive design for mobile and desktop devices.

### Technologies Used

- HTML
- CSS
- JavaScript

### How To Use

1. Open the `index.html` file in your web browser.
2. Enter a task in the input field and click the "Add Task" button.
3. Click the checkbox next to a task to mark it as complete.
4. Click the "Delete" button next to a task to remove it from the list.
